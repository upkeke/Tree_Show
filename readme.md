## QTimeLine控件
 是Qt中的一个时间线控件，可以用来实现动画效果。QTimeLine控件可以通过设置总时长、分步间隔和曲线类型来控制动画效果。你可以通过调用函数来控制动画的开始、暂停、停止等操作，并且还可以使用信号槽来监听动画状态的变化
